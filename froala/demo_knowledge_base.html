How to Set Up Your Account in 5 Easy Steps

üí° Tip: This guide will take approximately 5-10 minutes to complete. Have your email and phone number ready for verification.

Overview

Setting up your account is quick and easy. Follow these steps to get started with our platform and unlock all the features available to you.

Step-by-Step Instructions

Step 1: Create Your Account

  1. Navigate to the Sign Up page
  2. Enter your email address
  3. Choose a secure password (minimum 8 characters)
  4. Click "Create Account"
‚ö†Ô∏è Important: Make sure to use a valid email address as you'll need to verify it in the next step.

Step 2: Verify Your Email

Check your email inbox for a verification message and click the confirmation link.

Step 3: Complete Your Profile

  • Add your full name
  • Upload a profile picture (optional)
  • Set your timezone

Troubleshooting

If you encounter any issues during setup, please check our troubleshooting guide or contact support.

Next Steps

Once your account is set up, you can:

  • Explore the dashboard
  • Customize your settings
  • Start using our features

// Complete Knowledge Base Configuration
// Combines: Accessibility + Table Headers + Alt Text + Full Editing Features
new FroalaEditor('#kb-editor', {
  // Knowledge base specific toolbar
  toolbarButtons: {
    moreText: { buttons: ['bold','italic','underline','strikeThrough','subscript','superscript','fontFamily','fontSize','textColor','backgroundColor','clearFormatting'] },
    moreParagraph: { buttons: ['alignLeft','alignCenter','alignRight','alignJustify','formatOLSimple','formatUL','paragraphFormat','lineHeight','outdent','indent','quote'] },
    moreRich: { buttons: ['insertLink','insertImage','insertVideo','insertTable','emoticons','fontAwesome','specialCharacters','insertFile','insertHR'] },
    moreMisc: { buttons: ['undo','redo','fullscreen','print','getPDF','spellChecker','html','help'] }
  },
  
  // Mobile toolbar
  toolbarButtonsSM: {
    moreText: { buttons: ['bold','italic','underline','textColor'] },
    moreParagraph: { buttons: ['alignLeft','alignCenter','alignRight','formatOL','formatUL'] },
    moreRich: { buttons: ['insertLink','insertImage','insertTable'] }
  },
  
  // Knowledge base settings
  heightMin: 400,
  heightMax: 800,
  placeholderText: 'Write your help article content here...',
  charCounterCount: true,
  spellcheck: true,
  
  // Custom paragraph styles for knowledge base
  paragraphStyles: {
    'alert alert-info': 'Info Box',
    'alert alert-warning': 'Warning Box',
    'alert alert-success': 'Success Box',
    'alert alert-danger': 'Danger Box'
  },
  
  // Accessibility features
  tabIndex: 1,
  imageDefaultAlt: 'Knowledge base image',
  
  // Table configuration for accessibility
  tableStyles: {
    'table-bordered': 'Bordered Table',
    'table-striped': 'Striped Table'
  },
  tableDefaultWidth: '100%',
  
  // Events for accessibility and functionality
  events: {
    initialized: function () { console.log('KB editor initialized'); },
    contentChanged: function () {
      // Ensure table headers have proper scope
      this.$el.find('table tr:first th').attr('scope', 'col');
      // Ensure images have alt text
      this.$el.find('img').each(function() {
        if (!$(this).attr('alt')) {
          $(this).attr('alt', 'Knowledge base image');
        }
      });
    }
  }
});
