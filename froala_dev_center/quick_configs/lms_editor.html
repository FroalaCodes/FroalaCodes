Introduction to Variables

In algebra, a variable is a symbol (usually a letter) that represents an unknown value. Variables allow us to write general mathematical statements and solve problems systematically.

"Algebra is the intellectual instrument which has been created for rendering clear the quantitative aspects of the world." - Alfred North Whitehead

Key Concepts:

  1. Variables: Letters that represent unknown values (e.g., x, y, z)
  2. Constants: Fixed numerical values (e.g., 5, -3, π)
  3. Expressions: Combinations of variables and constants (e.g., 2x + 5)
  4. Equations: Mathematical statements showing equality (e.g., 2x + 5 = 13)

Example Problem

Problem: Solve for x: 2x + 5 = 13

Solution:

  1. Subtract 5 from both sides: 2x = 8
  2. Divide both sides by 2: x = 4

Answer: x = 4
